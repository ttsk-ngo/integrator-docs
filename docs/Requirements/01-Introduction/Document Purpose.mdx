# Document Purpose

The purpose of this document is to provide a detailed description of the Integrator 2.0 project. It will explain the purpose and features of the system, the system interfaces, what the system will do, the constraints within which it must operate, and how the system will respond to external stimuli. The document is intended for both stakeholders and system developers. The document is being created in an open-source format by volunteers and the TTSK team. The document is supervised by the TTSK association.
