# RS-UC0002 - Log View

|                            Element | Description                               |
| ---------------------------------: | :---------------------------------------- |
| Actor | user |
| Goal | Ability to display a list of all events registered in the system related to the user's reputation. |
| Precondition | 1. User is loggged in |
| Basic Flow | User selects the “Reputation System” icon from the left column, then selects the “Log” tab to view statistics.  |
| Alternative Flows | User in the “Global view” can select the “Log” icon available for each item. Selecting this icon redirects to the “Log” tab automatically narrowing the logs to the selected item. |

| Special Requirements |  |
