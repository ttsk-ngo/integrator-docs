# UM-FR0001 - List of the users

## General requirements

- List of the users should be visible to anyone, even if banned. 
- Technical and maintenance accounts should only be visible to users with administration roles.
- It should be possible to filter the list by username, role, and registration date.
- Is should be possible to navigate to a particular user's profile directly from the list.
- The list should provide a text search for usernames.
- The list should be limited to 30 users per page. 
- It should be possible to go directly to a DM with a user, provided they have registered with the TTSK chat provider.

## Authorization limitations

List of the users shouldn't be limited to any user role or claim. Technical and maintenence accounts should be hidden from public viewers.
