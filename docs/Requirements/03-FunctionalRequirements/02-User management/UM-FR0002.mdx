# UM-FR0002 - User profile

User profile should present following data:
- username
- profile image
- registration date
- last active date
- role of the user
- link to go to a DM with a user if they are registered with TTSK chat provider.

for users with moderation privileges following data should be presented as well:
- active sessions
- link to open user profile in moderation mode
