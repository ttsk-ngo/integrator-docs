# U-FR0002 - User badges

User badges are only Integrator-specific and are not shared across TTSK ecosystem. Their purpose is similar to roles, but they don't give access to any parts of the application.

Examples of user badges:
- Developer
- Pioneer
- Veteran
- Chief of moderators
- Board of directors
- etc.
