# U-FR0001 - User roles

Each of the roles grants access to different parts of the application. Roles corresond to the ones used by IdP.

Examples of roles:
- Administrator
- Moderator
- Team member
- User
- etc.

## Banned users
Users banned in Integrator app are handled by banning logic regardless of their roles.
